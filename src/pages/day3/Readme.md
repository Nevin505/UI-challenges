

# Day 3 - Hero Section Challenge

## Hero Section Layout

A hero banner section with dark teal background containing:
- Main heading text
- Descriptive paragraph text
- Call-to-action button

### Implementation Notes

- Hero section should have full-width dark teal background (`#23424A`)
- All text elements should be white/light colored for contrast
- Button should have rounded corners
- Button text should be uppercase
- Ensure responsive layout that works on all screen sizes
